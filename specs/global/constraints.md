# Global Constraints & Architecture

- **Stack**: React 18, Vite, TypeScript, Tailwind CSS.
- **Design Principles**: Premium, dynamic, immersive visual aesthetics (UI/UX first).
- **Motion**: Fallbacks obrigatórios para `prefers-reduced-motion`.
- **Data**: Foco estrito em casca visual (dummy data, ausência de integrações complexas de banco de dados neste escopo).
